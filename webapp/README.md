# Version Web de Budget Manager Pro V5

Ce dossier contiendra la version web de l'application, adaptée pour être déployée dans un conteneur avec Nginx.

## Étapes prévues
- Transformation de l'application en API web (Flask/FastAPI/Django)
- Création d'une interface web (frontend)
- Dockerisation (Dockerfile)
- Configuration Nginx
- Orchestration avec docker-compose

Tous les modules et fonctionnalités seront migrés pour être accessibles via le web.
